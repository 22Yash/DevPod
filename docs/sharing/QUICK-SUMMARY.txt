==================================================================
  TROUBLESHOOTING
==================================================================

Q: Share button not visible?
A: - Check workspace is "running" (not "stopped")
   - Check workspace is Python template
   - Refresh the page (F5)

Q: "Workspace is too large to share" error?
A: - Workspace exceeds 10MB limit
   - Remove large files
   - Delete __pycache__ folders

Q: Cloning fails?
A: - Check Docker is running
   - Check internet connection
   - Check backend logs
   - Verify you're logged in

Q: Packages not installed?
A: - Check requirements.txt exists
   - Check package names are valid
   - Check internet connectivity
   - Check backend logs for pip errors
