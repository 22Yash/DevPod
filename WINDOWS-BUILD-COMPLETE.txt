================================================================================
                        TROUBLESHOOTING QUICK FIXES
================================================================================

Problem: "Cannot find path"
Solution: Make sure you're in the right directory
  cd "C:\Users\Sangini\OneDrive\Desktop\devpod new\DevPod"

Problem: "Docker daemon not running"
Solution: Start Docker Desktop
  1. Click Windows Start
  2. Search "Docker Desktop"
  3. Click to open
  4. Wait 30 seconds
  5. Retry build

Problem: Build hangs
Solution: Wait longer (MERN can take 15 minutes)
  Or restart Docker and retry

Problem: "Out of disk space"
Solution: Free up space
  dir C:\
  Delete unnecessary files
  Empty Recycle Bin
  Retry build

Problem: Build fails
Solution: Try these steps
  1. Restart Docker Desktop
  2. Run: docker system prune -a
  3. Retry build
  4. If still fails, read DOCKER-BUILD-TROUBLESHOOTING.md
